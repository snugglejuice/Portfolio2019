Créer une carte de visite

Pour modifier les informations d’une carte de visite ou créer une nouvelle carte de visite, il faut modifier le fichier db.js qui se trouve dans Library > sceneSubject. Les cartes de visites sont représentées par des objets.

Pour créer une carte personnalisée, on ajoute les modules souhaitées dans le tableau option. Les modules existants jusqu'à maintenant sont :

  • profil :
  • Affiche une photo, le nom et le métier
  • nécessite les propriétés name, job et photoURL (String)

  • map
  • Affiche une carte centrée sur le lieu , l’adresse et la société
  • nécessite un objet map possédant les propriétés lat (float), lng  (float), adress (String)  et society (String)

  • about
  • Affiche un texte
  • nécessite un objet about possédant les propriétés title, text et image (String)

  • value
  • Affiche plusieurs textes qui se succèdent
  • nécessite l’objet value, possédant la propriété slideArray qui contient des slides, objet composé d’un id (int), title, text et image (String)

  • team
  • Affiche une photo
  • nécessite photoTeamURL (String)

  • project
  • Affiche une vidéo
  • nécessite videoURL (String)

  • client
  • Affiche une grille de photos
  • nécessite clientArray un tableau d’URL d’image ( Array de String )

Render / Update une carte

Utiliser la méthode render sur votre objet libraryCard

// Dans votre fonction servant à update votre scène ajouté :

libraryCard.render();

renderer.render(scene, camera);

Etape suivante

/!\     Ajouter de la lumière à votre scène car addCard() n’ajoute pas de lumière

Modifier les éléments d’une carte

Pour modifier les éléments d’une carte créer simplement une nouvelle carte en utilisant addCardFromArray(...); votre ancienne carte est supprimée et remplacée par la nouvelle carte avec les bon éléments

   libraryCard.addCardFromArray(id, position, scene, newOption, toIntersect);

/!\     Si une option est demandé mais qu’elle ne se trouve pas dans la db l’éléments est crée mais aucune information sera visible ou dans le cas des boutons ils ne seront pas cliquable

Modifier la position d’une carte

Pour modifier la position / rotation / size d’une carte, utiliser la methode setPosition en utilisant un objet position comme celui pour addCard

   libraryCard.setPosition(positionObj);
